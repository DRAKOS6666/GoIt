Задание 1

Создать 4 SASS-переменные которые хранят следующие значения:

  • набор шрифтов страницы: Roboto, sans-serif
  • цвет текста: #424242
  • цвет акцента при ховере: #03A9F4
  • размер шрифта абзацев: 18px

Result

          
    
            
  • $main-font-stack: 'Roboto', 'Arial', sans-serif;
    • 
            
  • $primary-dark-color: #424242;
    • 
            
  • $primary-accent-color: #03A9F4;
    • 
            
  • $main-text-size: 18px;
    •

Задание 2

Создать SASS миксин, который принимает 2 значения, размер шрифта и высоту строки. В место вызова должны подставляться font-size и line-height с передаными значениями.

Result

          
           
@mixin font-settings($size, $height) {
            font-size: $size;
            line-height: $height;
          }

          
.paragraph {
           @include font-settings(12px, 14px); 
          }
